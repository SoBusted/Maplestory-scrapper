# xCrawler
✂️ A crawling example for maplestory with various languages

## Run Code

### C#
```
dotnet run
```
or
```
mcs -out:Program Program.cs
./Program
```
---
### C++
```
make
./main
```
---
### Go
```
go run main.go
```
or
```
go build main.go
./main
```
---
### NodeJs
```
npm start
```
---
### Python
```
python app.py
```
