# xCrawler
✂️ A crawling example for maplestory with various languages

## Run Code

### C#
```
dotnet run
```
or
```
dotnet build -o Release -r ubuntu.18.04-x64 -c Release
./Release/Program
```
---
### C++
```
make
./main
```
---
### Go
```
go run main.go
```
or
```
go build main.go
./main
```
---
### Java
```
javac loadRank.java
java loadRank
```
---
### NodeJs
```
npm start
```
---
### Python
```
python app.py
```
